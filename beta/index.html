Beta Description

Glossier, Add to Cart Button, Image Slideshow, Show More Options Button
Link to the component

Trigger

Add to Cart Button

  • Hover on
  • Hover off

Image Slideshow

  • Hover on
  • Hover off

Show More Options Button

  • Hover on
  • Hover off
  • Mouse down
  • Mouse up

Rules

Add to Cart Button

Hover on

  • Static stroke on bottom and left switch to a moving dashed stroke with a 1.5px border width
  • The dashes animate and move around the perimeter in a clockwise motion
  • Cursor changes from arrow cursor to pointer.
  • Background becomes a dark gray
  • Call to action becomes underlined

Hover off

  • Reverts back to static stroke when no longer hovering
  • Cursor changes back from pointer to arrow cursor.
  • Background reverts back to white
  • Call to action loses underlined text decoration

Image Slideshow

Hover on

  • Images will change after 2 seconds when hovering, and will continue through all images in slideshow
  • Images fade over 0.3s
  • Mouse cursor becomes pointer

Hover off

  • Animation will stop and return back to the cover photo
  • Pointer cursor becomes mouse

Show More Options Button

Hover on

  • Static stroke switches to a moving dashed stroke
  • The dashes animate and move around the perimeter in a clockwise motion
  • Cursor changes from arrow cursor to pointer.

Hover off

  • Reverts back to static stroke when no longer hovering
  • Cursor changes back from pointer to arrow cursor.

Mouse down

  • Dashed moving stroke becomes solid and stops moving

Mouse up

  • Button disappears
  • Will show more options and make space for them under
  • Options fade in a down direction over 0.3s

Loops and Modes

Add to Cart Button

  • Loop: animation continues as long as the user is hovering on. Loop ends when user stops hovering.
  • No modes.

Images Slideshow

  • Loop: animation continues as long as the user is hovering on. Loop ends when user stops hovering.
  • No modes.

Show More Options Button

Loops

  • Animated dashed line continued to move clockwise until user clicks or hovers off

Modes

  • Clicking this button will introduce more options, adding 2 more boxes to the products shown
  • This button disappears at the same time, replaced by the new objects
