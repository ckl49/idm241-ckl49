Rules

Add to Cart Button

Hover on

  • Static stroke on bottom and right switch to a moving dashed stroke with a 1.5px border width
  • The dashes animate and move around the perimeter in a clockwise motion
  • Cursor changes from arrow cursor to pointer
  • Background becomes light gray
  • Text becomes underlined

Hover off

  • Reverts back to static stroke when no longer hovering
  • Cursor changes back from pointer to arrow cursor
  • Background reverts back to white
  • Call to action loses underlined text decoration

Mouse down

  • Animated border becomes a full solid border with a 1.5px width

Mouse up

  • Reverts back to animated stroke moving around clockwise with a 1.5px width

Image Slideshow

Hover on

  • Immediately switches to the 2nd image with a 0.8 second fade
  • After 1.5 seconds: Switches to the 3rd image with a 0.3 second fade
  • Every 1.5 seconds after that: Continues cycling through images all images until loops back around to 1
  • Mouse cursor becomes pointer

Hover off

  • Animation will stop and return back to the cover photo
  • Pointer cursor becomes mouse
  • Fade immediately over 0.3 second back to Image 1

Show More Options Button

Hover on

  • Static stroke switches to a moving dashed stroke
  • The dashes animate and move around the perimeter in a clockwise motion
  • Cursor changes from arrow cursor to pointer

Hover off

  • Reverts back to static stroke when no longer hovering
  • Cursor changes back from pointer to arrow cursor

Mouse down

  • Animated border becomes a full solid border with a 1.5px width

Mouse up

  • Button disappears
  • Becomes more options and make space for them under
  • Options fade in over 0.8 seconds

Loops and Modes

Add to Cart Button

  • Loop: animation continues as long as the user is hovering on. Loop ends when user stops hovering or puts mouse down, but then continues once mouse is up.
  • No modes.

Images Slideshow

  • Loop: animation continues as long as the user is hovering on. Loop ends when user stops hovering.
  • No modes.

Show More Options Button

Loops

  • Animated dashed line continued to move clockwise until user clicks or hovers off

Modes

  • Clicking this button will introduce more options, adding 2 more boxes to the products shown
  • This button disappears at the same time, replaced by the new objects
  • Introduces two new colored boxes in its place, wrapping around to the next line and extending the size of the frame
