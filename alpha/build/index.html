Link to the component

Trigger

Add to Cart Button

  • Hover on
  • Hover off

Rules

Add to Cart Button

Hover on

  • Hover on: static stroke switches to a moving dashed stroke
  • Hover on: The dashes animate and move around the perimeter in a clockwise motion
  • Hover on: Cursor changes from arrow cursor to pointer.

Hover off

  • Hover off: Reverts back to static stroke when no longer hovering
  • Hover off: Cursor changes back from pointer to arrow cursor.

Add to Cart Button
