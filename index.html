    Casey Lee
    Glossier Button
    Link to the component:

    Trigger

    • User hovers on and off the button

    Rules

    • Static stroke switches to a moving dashed stroke
    • The dashes move around the perimeter in a clockwise motion
    • Reverts back to static stroke when no longer hovering

    Feedback

    • The static, solid stroke disappears
    • The stroke becomes dashed and covers the entire perimeter of the button
    • The stroke moves physically around the button clockwise, to outline the shape

    Loops and Modes

    • Loop: the dashed stroke animation continues for as long as the user is hovered, and is stopped/reset when the user stops hovering.
    • No modes.
